# Lab12_Fernando_Mas Implementacion con Cupertino

Este laboratorio implementa el mismo contenido del Lab08, pero reemplazando los componentes de Material por componentes de Cupertino para dar una apariencia más cercana a iOS.

## Tecnologias y Componentes Utilizados

- Flutter con Dart.
- `CupertinoApp` como estructura principal de la aplicacion.
- `CupertinoPageScaffold` para las pantallas.
- `CupertinoNavigationBar` para la barra superior.
- `CupertinoButton` para botones de accion.
- `CupertinoTextField` para campos de texto.
- `CupertinoActivityIndicator` para la pantalla de carga.
- `CupertinoPicker` para seleccionar categorias.
- `CupertinoDatePicker` para seleccionar fecha de vencimiento.
- `CupertinoIcons` para iconografia estilo iOS.

## Funcionamiento de la App

La aplicacion simula una bodega llamada BodegaFMAS. Permite iniciar sesion, mostrar una carga breve, navegar por un menu principal, registrar productos, seleccionar categoria y fecha con controles Cupertino, ver un listado de productos y editar el perfil del usuario.
